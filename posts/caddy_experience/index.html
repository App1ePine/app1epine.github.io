Caddy Docs

软件介绍

Caddy 是一个支持 HTTP/2 的跨平台 Web 服务器，使用和配置都非常简单。Caddy 支持 HTTP/2, IPv6, Markdown, WebSockets, FastCGI, 模板等等。

软件特性

  • 全自动支持HTTP/2协议
  • 使用 Let’s Encrypt 全自动部署HTTPS
  • 支持 IPv6
  • 较好支持 WebSockets
  • 自动将 Markdown 转化为 Html
  • 简易定义 log
  • 容易部署，Go 只是一个二进制文件，没有依赖
  • 支持三大系统，得益于 Go 的跨平台特性

在 Caddy 中，所有的功能都是插件，HTTP 服务是插件，TLS 服务是插件，每一行命令的实现都是插件

运行 caddy

推荐通过修改配置文件 Caddyfile 方式运行 caddy

1
2
3
4

➜  ~ cd /etc/caddy 
➜  caddy ls
Caddyfile
➜  caddy vim Caddyfile

配置样例：

1
2
3

:8080 {
    respond "Hello World!"
}

运行 caddy：

1

caddy start

出现 Successfully started Caddy (pid=33672) - Caddy is running in the background 即 Caddy 启动成功

可以通过 Curl 请求上述配置的端口检验是否正常运行

1

curl http://127.0.0.1:8080

返回 “Hello World”即为配置正常，也可以直接访问上述页面

部署静态网站

假设我们的静态网站或者 HTML 文件放在 /home/你的用户名/sites/demo 下，修改配置文件

1
2
3
4
5

# v2 版本 caddy
domain:888 {
	root * /home/你的用户名/sites/demo
	file_server browse
}

例如我的博客本地部署的配置：

1
2
3
4

http://127.0.0.1:3003 {
        root * /home/ApplePine/Desktop/blog/docs
        file_server browse
}

更多详细配置方式请查看官方文档
